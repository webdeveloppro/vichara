<h1>Vichara</h1>
<h2>How to install project</h2>
<p> git clone git@github.com:webdeveloppro/vichara.git </p>
<p> yarn install or npm install </p>
<p> Start project:</p>
<p> yarn gulp or npm gulp</p>

<p>
folder pages -> in this folder you can create files for page
<br>
_fonts -> import font from google font
<br>
_libs -> file for foundation components
<br>
_vars -> vars for color and fonts
<br>
main -> here need import all components
</p>
